Rock Paper Scissors Game 🎮

Project Description

This project is a simple Rock Paper Scissors game developed using HTML, CSS, and JavaScript.
It allows a user to play the classic Rock-Paper-Scissors game against the computer. The computer randomly selects a move, and the result (win, lose, or draw) is displayed on the screen along with the updated score.

Technologies Used

- HTML – Used to create the structure of the game interface.
- CSS – Used to style the layout, buttons, and result messages.
- JavaScript – Used to implement the game logic and handle user interactions.

Game Rules

The game follows the standard Rock-Paper-Scissors rules:

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
- If both choices are the same, the game results in a draw.

Features

- Interactive user interface
- Clickable icons for Rock, Paper, and Scissors
- Random computer move generation
- Score tracking for both the user and the computer
- Display messages for win, lose, or draw
- Simple and responsive design

How to Run the Project

1. Download or clone the project files.
2. Open the project folder.
3. Double-click the index.html file.
4. The game will open in your web browser.
